# DevTinder
---
DevTinder is a feature-rich MERN Stack application that brings the concept of modern connection-building to the digital space. Tailored for developers, it enables users to explore, connect, and manage their professional network seamlessly.

---


## 🌟 Features

- **User Authentication**: Seamless signup, login, and logout functionality.
- **Profile Management**: View and edit user profiles with ease.
- **Connection Requests**:
  - Send requests to connect with other users.
  - Review requests (accept, reject, or ignore).
- **Received Requests**: View all incoming connection requests.
- **Accepted Connections**: Track all approved connections.
- **User Feed**: Explore profiles of other users on the platform.
- **Subscription .**:User only send message with subscription

---

## 🛠️ Tech Stack

### Frontend
- **React.js**: For building dynamic and responsive user interfaces.
- **Redux Toolkit**: For state management.
- **Axios**: For handling API requests.
- **daisyUI**: Tailwind css Library.
  

### Backend
- **Node.js** with **Express**: For building a robust and scalable backend API.

### Database
- **MongoDB**: For efficient data storage and retrieval.

### Authentication
- **JWT**: For secure user sessions.

## 🚀 Installation Guide

Follow these steps to set up the project locally:

### Prerequisites

- Node.js installed on your machine
- MongoDB set up locally or using MongoDB Atlas
- Git installed

### Steps

1. **Clone the Repository**
   ```bash
   git clone https://github.com/sharmaji513/devtinder.git
   cd devtinder
   ```

2. **Install Dependencies**

   - For the backend:
     ```bash
     cd backend
     npm install
     ```

   - For the frontend:
     ```bash
     cd frontend
     npm install
     ```

3. **Set Up Environment Variables**

   - Create a `.env` file in the `backend` folder and add the following:
     ```env
     PORT=5000
     MONGO_URI=<your_mongo_db_connection_string>
     JWT_SECRET=<your_jwt_secret>
     ```

4. **Run the Application**

   - Start the backend:
     ```bash
     cd backend
     npm start
     ```

   - Start the frontend:
     ```bash
     cd frontend
     npm start
     ```

5. **Access the Application**

   Open your browser and navigate to `http://localhost:3000`.
